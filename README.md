# vfox-nginx-exporter

[![CI](https://github.com/jbox-web/vfox-nginx-exporter/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-nginx-exporter/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [nginx-prometheus-exporter](https://github.com/nginx/nginx-prometheus-exporter)

## Install

```
mise plugin install nginx-exporter https://github.com/jbox-web/vfox-nginx-exporter.git
```
